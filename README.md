# get-oui
Gets the vendor reference from organizational unique identifiers and MAC addresses.

Supports numerous formats such as:

`xx:xx:xx:xx:xx:xx`<br>
`xx.xx.xx.xx.xx.xx`<br>
`xx-xx-xx-xx-xx-xx`<br>
`xxxx-xxxx-xxxx`<br>
`xx-xx-xx`<br>
`xx.xx.xx`<br>
`xx:xx:xx`<br>
`xxxx-xx`<br>
